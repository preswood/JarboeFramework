## JarboeFramework
JarboeFramework is the direct successor of Jarboe (AKA JarboeFX), which was abandoned in 2024.
The biggest improvements over JarboeFX include an overhauled naming scheme for classes and more importantly switching compliancy to Java 5 (Windows 9x and Mac OS X 10.0 are now supported).

# How to install
1. Download the zip file of the latest release
2. Extract the downloaded zip
3. Create a "jarboe" package in your IDE
4. Drag n drop or import the files into the "jarboe" package
